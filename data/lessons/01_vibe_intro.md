# AI是你的超级队友

---

## 想象一下...

你有一个朋友，他：
- 24小时在线，随叫随到
- 什么都懂一点，从写作业到编程
- 打字超快，几秒钟就能写一大段
- 从不嫌你问题多，永远有耐心

这个朋友，就是 **AI**。

今天开始，你要学会怎么跟这个超级队友合作，让它帮你做出超酷的东西！

---

## 马上试试！

在下方的对话框里，发送这句话：

```
你好，你是谁？你能帮我做什么？
```

看看AI怎么回答你。

---

## 让AI秀一下

AI能做的事情超乎你想象，试试这些：

**让它画个画：**
```
用文字符号画一只小猫
```

**让它讲个故事：**
```
用50个字讲一个关于勇气的小故事
```

**让它出个谜语：**
```
出一个关于西瓜的谜语，给我3次猜的机会
```

**让它当翻译：**
```
"我爱编程"用英语、日语、韩语怎么说？
```

是不是挺有意思的？这只是AI能做的一小部分！

---

## AI翻车时刻

不过，AI也不是万能的。来看几个它会翻车的场景：

**翻车1：数学计算**

试试发这个：
```
计算 17 × 23 + 456 ÷ 8
```
然后自己用计算器验算一下，看AI算对了没？

**翻车2：实时信息**

试试问：
```
今天天气怎么样？
```
AI知道吗？为什么？

**翻车3：编故事当真事**

试试问：
```
李白写过哪些关于手机的诗？
```
看看AI会不会一本正经地胡说八道。

---

## 划重点：AI的特点

**AI很厉害：**
- 写文章、改文章
- 翻译各种语言
- 回答知识问题
- 帮你想点子、出主意
- 写代码（这门课的重点！）

**AI有局限：**
- 数学计算可能出错（它是语言模型，不是计算器）
- 不知道"现在"发生的事
- 有时候会编造不存在的东西
- 它说的话不一定对，要自己判断！

**记住这句话：AI是队友，不是答案机。它帮你干活，但你得动脑子。**

## 探索任务

完成这3个任务，看看你能发现什么：

**任务1：让AI帮你写自我介绍**

发送：
```
帮我写一段自我介绍，我是初中生，喜欢[填你的爱好]
```
看看AI写得怎么样？哪里需要改？

**任务2：故意让AI翻车**

想一个问题，让AI回答不了或者回答错。
比如问它"你今天吃了什么"或者"明天会下雨吗"。

**任务3：让AI教你一个东西**

选一个你想学的东西，让AI用最简单的话解释：
```
用最简单的话告诉我什么是[你想学的东西]，就像跟小学生解释一样
```

## 这门课要做什么？

接下来的课程，你会用AI做出这些东西：

- 一个能玩的**小游戏**（贪吃蛇）
- 一个实用的**小工具**（番茄钟、单词卡片等）
- 一个属于你的**个人网站**（可以发到网上让别人看！）

**不用背代码，不用学复杂语法。**

你只需要学会一件事：**怎么跟AI说清楚你想要什么。**

这就是这门课最重要的技能。

---

## 划重点！

1. AI是你的超级队友，24小时在线，啥都能帮忙
2. 但AI也会翻车：算数可能错、不知道实时信息、有时会编故事
3. AI说的话要验证，它是帮手不是权威
4. 这门课教你用AI做游戏、工具、网站
5. 核心技能：**学会跟AI说清楚你要什么**
