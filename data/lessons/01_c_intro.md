# 第一章：C语言入门

## 1.1 什么是C语言

C语言是一门**通用、过程式**的编程语言，由Dennis Ritchie于1972年在贝尔实验室开发。它是最古老也是最有影响力的编程语言之一。

### C语言的特点

- **高效**：接近硬件，运行速度快
- **可移植**：可在不同平台编译运行
- **功能强大**：支持底层操作
- **应用广泛**：操作系统、嵌入式系统、驱动程序等

## 1.2 第一个C程序

让我们来写一个经典的"Hello World"程序：

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

### 代码解析

| 代码 | 含义 |
|------|------|
| `#include <stdio.h>` | 包含标准输入输出库 |
| `int main()` | 主函数，程序入口 |
| `printf()` | 输出函数 |
| `\n` | 换行符 |
| `return 0;` | 返回0表示程序正常结束 |

## 1.3 C程序的编译过程

C语言是**编译型语言**，需要经过以下步骤：

1. **预处理**：处理 `#include`、`#define` 等指令
2. **编译**：将C代码转换为汇编代码
3. **汇编**：将汇编代码转换为机器码
4. **链接**：将多个目标文件链接成可执行文件

```bash
gcc hello.c -o hello    # 编译
./hello                  # 运行
```

## 1.4 基本语法规则

### 语句和分号

每条语句以**分号 `;`** 结束：

```c
printf("Hello");
int a = 10;
```

### 注释

```c
// 这是单行注释

/*
   这是
   多行注释
*/
```

### 代码块

使用**花括号 `{}`** 包围代码块：

```c
int main() {
    // 代码块内容
}
```

## 1.5 printf 输出函数

`printf` 是最常用的输出函数：

```c
#include <stdio.h>

int main() {
    printf("我正在学习C语言\n");
    printf("C语言很有趣！\n");
    return 0;
}
```

### 转义字符

| 转义字符 | 含义 |
|----------|------|
| `\n` | 换行 |
| `\t` | 制表符（Tab） |
| `\\` | 反斜杠 |
| `\"` | 双引号 |

## 本章小结

- C语言是高效、可移植的编译型语言
- 每个C程序都从 `main()` 函数开始执行
- 语句以分号结束
- 使用 `printf()` 进行输出
- 代码需要先编译再运行
