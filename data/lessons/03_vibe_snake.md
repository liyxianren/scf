## 30秒了解网页

在开始之前，先花30秒了解一下我们要用的技术。

网页是由3种东西组成的：

| 名字 | 干嘛的 | 比喻 |
|------|--------|------|
| HTML | 放内容 | 房子的结构 |
| CSS | 弄好看 | 房子的装修 |
| JavaScript | 能动起来 | 房子里的电器 |

贪吃蛇游戏需要"能动起来"，所以我们会用到 HTML + JavaScript。

**不用背这些**，AI会帮你写代码。你只需要知道它们是什么就行。

---

## 先想清楚：贪吃蛇怎么玩？

做游戏之前，先想清楚规则：

1. 有一条蛇，会自己往前跑
2. 用方向键（↑↓←→）控制蛇的方向
3. 地图上有食物，蛇吃到食物会变长
4. 吃一个食物，分数+1
5. 蛇撞到墙，或者撞到自己，游戏结束
6. 游戏结束后可以重新开始

就这些！现在把规则告诉AI。

## 写Prompt，让AI开工

把下面的内容复制，发给AI：

```
你是一个游戏开发高手。

请帮我做一个贪吃蛇游戏。

要求：
- 用 HTML + JavaScript，所有代码放在一个文件里
- 游戏窗口 400 x 400 像素，背景是深灰色
- 蛇是绿色的方块
- 食物是红色的圆点
- 用键盘方向键控制
- 吃到食物蛇变长，分数+1
- 撞墙或撞到自己就游戏结束
- 游戏结束显示最终分数，有"重新开始"按钮

效果类似于谷歌搜索"贪吃蛇"出来的那个小游戏。
```

发完等AI回复。它会给你一大段代码。

## 运行你的游戏

拿到代码后，按这几步操作：

**第1步：复制代码**

AI回复的代码，全部选中，复制。

**第2步：新建文件**

在桌面上新建一个文本文件，把后缀名改成 `.html`。

比如叫 `snake.html`

**第3步：粘贴代码**

打开这个文件（用记事本或其他文本编辑器），把代码粘贴进去，保存。

**第4步：打开玩！**

双击 `snake.html`，它会在浏览器里打开。

开始玩吧！

## 常见翻车急救站

游戏跑不起来？别慌，看看是不是这些问题：

**问题1：双击文件打开后显示一堆代码，不是游戏**

原因：用记事本打开了，不是用浏览器打开。

解决：右键点击文件 → "打开方式" → 选择 Chrome/Edge/Firefox 浏览器。

**问题2：方向键按了没反应**

原因：焦点不在游戏上。

解决：用鼠标点一下游戏画面，再按方向键。

**问题3：蛇不动/游戏没画面**

原因：代码可能没复制完整，或者有错误。

解决：把你看到的问题告诉AI，让它帮你修。比如发：
```
我的贪吃蛇游戏打开后蛇不动，代码如下：
[粘贴你的代码]
请帮我检查哪里出了问题
```

**问题4：中文显示乱码**

解决：让AI在代码开头加上 `<meta charset="UTF-8">`
