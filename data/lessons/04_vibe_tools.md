# 做一个属于你的小工具

---

## 你有什么烦恼？

先别急着写代码，想一想：

- 背单词老是记不住？
- 写作业总是走神？
- 每天的任务容易忘？
- 算数学题太慢？

**这些烦恼，都可以用一个小工具来解决。**

而且，用AI帮忙，做一个小工具可能只需要20分钟！

---

## 选一个来做

这里有几个推荐的小工具，选一个你最想做的：

| 工具 | 解决什么问题 | 难度 |
|------|--------------|------|
| 番茄钟 | 专注学习，定时休息 | 简单 |
| 计算器 | 算数学题 | 简单 |
| 单词卡片 | 背单词 | 中等 |

如果你选了其他的，方法是一样的！

---

## 什么是番茄钟？

番茄钟是一种学习方法：

1. 专注学习 25 分钟
2. 休息 5 分钟
3. 重复，每4个番茄休息长一点

很多人用这个方法提高效率。现在，你要自己做一个！

---

## 自己想一想：番茄钟需要什么？

在问AI之前，先自己想一想：

一个番茄钟应该有什么功能？

试着在脑子里列个清单，比如：
- 要显示倒计时
- 要有开始/暂停按钮
- 时间到了要提醒
- ……

想好了？现在来写Prompt。

## 写你自己的Prompt

这次，不直接给你现成的Prompt。

根据你想到的功能，试着自己写一个。

提示：还记得上一章说的4件事吗？
1. 让AI扮演什么角色？
2. 你要它做什么？
3. 有什么具体要求？
4. 有没有参考例子？

**自己试着写完再往下看！**

---

## 参考Prompt

这是一个参考，你的不一定要和这个一样：

```
你是一个前端开发高手。

请帮我做一个番茄钟应用。

功能要求：
- 默认工作25分钟，休息5分钟
- 大大的倒计时显示（分:秒）
- 有开始、暂停、重置按钮
- 工作时间结束自动开始休息
- 时间切换时发出提示音
- 显示当前是\"工作中\"还是\"休息中\"

界面要求：
- 工作时背景是红色系
- 休息时背景是绿色系
- 时间数字要大，好看清

请把所有代码放在一个HTML文件里。
```

你写的和这个差多少？不管怎样，发给AI看看效果！

---

## 运行你的工具

拿到AI给的代码后：

1. 复制全部代码
2. 新建文件，命名为 `pomodoro.html`
3. 粘贴代码，保存
4. 用浏览器打开

你的番茄钟就做好了！

## 验收检查

检查一下你的番茄钟：

- [ ] 能显示倒计时
- [ ] 点开始能开始计时
- [ ] 点暂停能暂停
- [ ] 点重置能重新来
- [ ] 25分钟到了会自动切换到休息
- [ ] 有提示音或颜色变化

## 不满意？改它！

工具跑起来了，但可能不完全符合你的想法。

**这时候就是学习的好机会！**

想想你还想加什么，或者想改什么，告诉AI。

比如：

**改成你喜欢的时间：**
```
请把工作时间改成30分钟，休息时间改成10分钟
```

**加上完成次数：**
```
请加一个计数器，记录今天完成了几个番茄
```

**改成你喜欢的颜色：**
```
请把工作时的背景改成蓝色，休息时改成紫色
```

## 做其他工具

如果你选的不是番茄钟，可以参考这些思路：

**计算器的要点：**
- 有数字按钮0-9
- 有加减乘除按钮
- 有等于按钮和清除按钮
- 能显示输入和结果

**单词卡片的要点：**
- 正面显示英文
- 点击翻转显示中文
- 有下一个按钮
- 最好能标记已经会的

**Todo清单的要点：**
- 能添加新任务
- 能标记任务完成
- 能删除任务
- 最好能保存（刷新不丢失）

写Prompt的时候，把这些要点都说清楚！

## 挑战任务

**挑战1：做一个你自己需要的工具**

不一定是上面列的这些。想想你平时有什么需求，能不能用一个小网页解决？

比如：
- 随机选择器（今天吃什么）

- 成绩计算器

想到什么就试着做做看！

在你的工具里藏一个彩蛋，比如：
- 点击某个地方会出现一行字
- 完成某个条件会放烟花

告诉AI你的想法，看它能不能帮你实现！
